# Expense Tracker — Трекер расходов

Графическое приложение для учёта личных расходов с фильтрацией, подсчётом
суммы за период, сохранением в JSON и тестами.

**Автор: Кувайцев Тимофей**

---

## Возможности

- Форма добавления расхода: сумма, категория, дата
- Подсчёт итоговой суммы расходов (по всем записям или по фильтру)
- Фильтрация по категории и по дате
- Сохранение данных в `data.json` и загрузка при следующем запуске
- Валидация: сумма — положительное число, дата — формат YYYY-MM-DD

---

## Структура проекта

```
expense-tracker/
├── main.py          # GUI-приложение (tkinter)
├── validator.py     # Валидация суммы и даты
├── storage.py       # Сохранение / загрузка данных (JSON)
├── tests.py         # Юнит-тесты (unittest)
├── requirements.txt
└── .gitignore
```

---

## Установка и запуск

### Windows

```bash
python main.py
```

### Linux

```bash
sudo apt install python3-tk
python3 main.py
```

---

## Запуск тестов

```bash
python -m unittest tests -v
```

14 тестов покрывают:
- валидацию суммы и даты
- сохранение / загрузку JSON
- подсчёт суммы расходов
- логику фильтрации

---

## Примеры использования

1. Запустить `python main.py`
2. Ввести сумму `250`, выбрать категорию `еда`, дату `2025-05-01`
3. Нажать **Добавить расход** — запись появится в таблице
4. Внизу отображается итоговая сумма
5. Использовать фильтр для просмотра расходов по категории или дате
